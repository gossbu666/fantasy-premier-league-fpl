# Fantasy Premier League Prediction System

Machine learning framework for predicting FPL player points using ensemble methods.

## Project Structure
- `src/feature_engineering/`: Feature building modules
- `src/models/`: Model training and optimization
- `src/evaluation/`: Prediction and evaluation scripts
- `src/utils/`: Utility functions
- `notebooks/`: Jupyter notebooks for analysis
- `results/`: Figures and metrics
